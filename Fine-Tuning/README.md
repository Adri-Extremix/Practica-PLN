# Fine-Tuning

Espacio reservado para los experimentos supervisados de HOPE-EXP.

La idea es reutilizar la base comun en `shared/` para:

- lectura y normalizacion del dataset,
- EDA inicial,
- esquemas comunes de etiquetas y anotaciones.

Cuando llegue el dataset oficial, se podran construir aqui los scripts de entrenamiento por subtarea sin duplicar la capa de carga de datos.
